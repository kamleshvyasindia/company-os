# Daily Ritual Prompts (Manual Cron Runner)

Since there is no automated background cron scheduler running in this local repository workspace, you (the founder) can run the minimal automation set manually using these daily prompts in your coding assistant.

---

## Morning Ritual (Start of Day)

Paste this prompt to run the morning brief:

```text
Read skills/morning-brief.md and execute it to generate today's morning brief in logs/briefs/.
```

---

## Evening Ritual (End of Day)

Paste this prompt to file inbox items and review the day's logs/failures:

```text
1. Read and execute skills/inbox-triage.md to file all new inbox captures into the company brain.
2. Read and execute skills/session-review.md to review today's logs, update the changelog, and optimize our skills.
```

---

## Meeting Capture Ritual (Per Meeting)

Paste this prompt after a key sync/meeting to log decisions:

```text
Read skills/meeting-capture.md and execute it using the meeting transcript or summary I provide.
```
