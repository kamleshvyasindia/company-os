# inbox/ — capture lands here. Append-only.

Everything worth remembering enters the company here, with zero filing decisions: meeting transcripts, voice-note transcriptions, screenshots, pasted Slack threads, links, half-formed ideas.

**Rules:**
- File naming: `YYYY-MM-DD-short-slug.md` (agents: generate the slug; humans: don't overthink it).
- **Append-only.** Never edit, rename, or delete an existing inbox file — not even to "clean up." The nightly `inbox-triage` skill files content into the brain; the original stays here as the raw record.
- If it's in someone's head or a WhatsApp thread, it doesn't exist. Get it in here.

The whole capture habit survives on one property: putting something in requires **no decisions**. Triage is the system's job, not yours.
