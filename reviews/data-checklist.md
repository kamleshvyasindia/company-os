# Data gate (numbers, classifications, reports)

- [ ] Every metric verified against its primary source (not a cached or joined copy)
- [ ] Definitions stated (what counts as "active user"?) — mismatched definitions are the #1 silent error
- [ ] Sample spot-check done on any bulk classification (≥5 items)
- [ ] Stale-data check: when was the source last updated?
- [ ] Uncertainty stated, not hidden — "approximately" beats confidently wrong
